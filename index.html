Add New Task

Owner can assign to anyone. Manager can assign to team. Staff can add self-task only.

Workflow Rules

  1. New assigned tasks start as Pending; staff self-tasks start as Waiting Approval until approved.
  2. Staff can update a task and use Sent for Review to send finished work for review.
  3. The same Task ID can become Completed or Needs Correction. No new Task ID is created for correction.
  4. Manager can act only on own-team tasks.

Update Task
