Add New Task

Owner can assign to anyone. Manager can assign to team. Staff can add self-task only.

Workflow Rules

  1. Owner/Manager tasks start as Pending; staff self-tasks start as Requested until approved.
  2. Staff can update only Pending / In Progress / Ready for Check.
  3. Requested tasks can be approved to Pending or Cancelled; Ready for Check tasks can become Completed or Revision Required.
  4. Manager can act only on own-team tasks.
  5. Owner manages people and login PINs from Admin / People.
  6. Owner can delete wrong tasks and remove staff/manager records. If a person has task history, the app safely deactivates them instead of breaking history.
